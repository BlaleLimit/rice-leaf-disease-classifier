This folder contains the Jupyter Notebooks responsible for testing, and creating preprocessing methods to be used on the images of the dataset.

GENERATED DATASET - contains the generated test datasets created by using the different preprocessing methods.

It contains 3 methods:
1. Generate Dataset for 4 Classes (Hispa, Healthy, BrownSpot ,LeafBlast)
2. Generate Dataset for 2 Classes (Disease, Healthy)
3. Generate Dataset for 3 Classes (Brown spot, Bacterial leaf blight, Leaf Smut)
